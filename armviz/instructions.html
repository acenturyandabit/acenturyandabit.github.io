How to use:

  1. Compile and run your program, with some sample data.
  2. Copy the OUTPUT of your program (i.e. the 3 or 5 numbered rows) into the big textbox.
  3. Copy the run command (e.g. ./robot 1 1 2 2 3 3 -r -i) into the run command textbox under the big textbox.
  4. OR, you can set the parameters in the boxes below.
  5. Set the display settings: Scale makes it larger, Grid step changes the increments of the grid
  6. Hit RUN. Enjoy.

Disclaimer: This visualiser WILL NOT -

  • Detect invalid arm positions, but if the arm lies outside the visible field, it lies outside the box.
  • Display the initial conditions from your runcommand, because that's part of the assignment.
  • Do your assignment for you!

But you're smarter than it. I believe in you!
